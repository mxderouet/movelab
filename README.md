# MoveLab

Production reference tool for the Ableton Move. Genre-specific drum patterns, bass lines, track suggestions, song structure guides, and workflow tips — all in one place.

## Genres

- House
- Techno
- Dub
- Drum'n'Bass

## Features

- 16-step drum sequencer with editable cells and pattern presets
- Bass step grid with note names, matching the drum layout
- Track suggestions for tracks 3 and 4 (harmony, melody, texture)
- Song structure timeline per genre
- Workflow and tips for Ableton Move

## Stack

- React + Vite
- No dependencies beyond React

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).
